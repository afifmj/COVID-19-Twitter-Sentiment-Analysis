This is a sentiment analyser of tweets regardijng COVID-19. The tweets can be worlwide or region specific as per the users needs. 
The sentiments are broadly classified as positive, negative, and neutral. 
The sentiment analysis is done using the TextBlob module and the tweets are collected from Twitter using tweepy.

This project was a challenge of Siraj Raval's "Learn Python for Data Science" series.
